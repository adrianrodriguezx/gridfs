1. Create a free sanbox database with https://mlab.com/
2. Create a new database user
3. Replace the mongoDB_URI value with your own mlab MongoDB URI
4. Place a video file inside readFrom
5. Update the videoPath value in storeVideo.js 
6. $ npm i
7. $ node storeVideo.js
8. $ node retrieveVideo.js
